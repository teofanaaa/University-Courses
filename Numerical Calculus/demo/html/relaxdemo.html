Initializare

A = [2,1;5,7];
b = [4;19];
xn = [2;1];

Pregatirea matricelor metodei

omega = 1; %optim 1.11
D = diag(diag(A)); L=-tril(A,-1);
M = 1/omega*D-L
N = M -A

M =

     2     0
     5     7


N =

     0    -1
     0     0

Prima iteratie

xv = xn;
xn = M\(N*xv+b);
ea = norm(xn-xv,inf);
